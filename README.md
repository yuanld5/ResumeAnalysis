# ResumeAnalysis

简历分析项目

## 环境设置

### 1. 创建虚拟环境

```bash
python -m venv venv
```

### 2. 激活虚拟环境

**macOS/Linux:**
```bash
source venv/bin/activate
```

**Windows:**
```bash
venv\Scripts\activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

## 依赖包

- langgraph: 用于构建语言模型应用
- unstructured: 用于解析非结构化文档
- langchain: LangChain核心库
- langchain-community: LangChain社区集成

## 使用

项目初始化完成，可以开始开发。
